# AI 配置

AI 功能是可选的增强模块。不配置 AI 时，你仍然可以正常使用监控和提醒功能。

## 启用 AI

1. 在侧边栏进入「AI 配置」页面
2. 打开"AI 增强"开关
3. 填写**业务描述** — AI 据此判断帖子相关性并生成回复
4. 选择**回复风格**（专业 / 温和 / 幽默 / 直接 / 共情），可选填回复模板
5. 选择模型供应商，填写模型名称、接口地址
6. 保存 API Key（存入系统 Keychain）
7. 点击"测试连接"确认配置正确
8. 保存配置

## AI 做了什么

启用后，软件在抓取到新帖子后会额外处理：

1. **基础过滤** — 去除平台官方账号发布的内容、商家广告、与业务无关的帖子
2. **相关性判断** — 基于帖子内容和你的业务描述判断是否值得提醒
3. **摘要提炼** — 生成帖子主题、痛点和关注点摘要
4. **回复样本生成** — 结合你的业务描述和回复风格，生成回复样本

## 密钥安全

API Key 存储在系统 Keychain（macOS）或 Credential Manager（Windows）中，不会出现在配置文件或日志中。

软件使用你自己的 API Key 直接调用模型服务，不经过任何第三方中转。

## 支持的模型

支持兼容 OpenAI 风格 API 的模型服务，包括但不限于：

- 智谱 AI（glm-4-flash 系列，有免费额度）
- DeepSeek
- OpenAI（GPT-4.1 系列）
- Moonshot
- 其他兼容 OpenAI Chat Completions 接口的服务
