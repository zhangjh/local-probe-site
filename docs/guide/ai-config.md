# AI 配置

AI 功能是可选的增强模块。不配置 AI 时，你仍然可以正常使用监控和提醒功能。

## 启用 AI

1. 在配置页面打开"AI 增强"开关
2. 填写以下信息：
   - **模型供应商** — 如 openai、deepseek 等
   - **模型名称** — 如 gpt-4.1-mini、deepseek-chat
   - **接口地址** — 兼容 OpenAI 格式的 API 地址
   - **API Key** — 你的模型服务密钥
3. 点击"测试连接"确认配置正确
4. 保存配置

## AI 做了什么

启用后，软件在抓取到新帖子后会额外处理：

1. **基础过滤** — 去除明显广告、水贴、低相关内容
2. **相关性判断** — 基于帖子内容和关键词判断是否值得提醒
3. **摘要提炼** — 生成帖子主题、痛点和关注点摘要
4. **回复样本生成** — 结合你的业务信息，生成 2-3 条不同风格的回复样本

## 密钥安全

API Key 存储在系统 Keychain（macOS）或 Credential Manager（Windows）中，不会出现在配置文件或日志中。

软件使用你自己的 API Key 直接调用模型服务，不经过任何第三方中转。

## 支持的模型

首发支持兼容 OpenAI 风格 API 的模型服务，包括但不限于：

- OpenAI（GPT-4.1 系列）
- DeepSeek
- 其他兼容 OpenAI Chat Completions 接口的服务
